Duplicate Files Cleanup Utility - Instructions

Getting Started

This utility helps you find and manage duplicate files in your system. It provides an intuitive web interface powered by Streamlit for easy duplicate file management.

Using the Software

1. Configuration (Sidebar)

  • Minimum file size: Set the smallest file size to consider (in KB)
  • File types: Specify which file extensions to scan (e.g., .txt,.pdf)
  • Exclude directories: List directories to skip during scan
  • Scan Options:
    • Scan hidden files: Include or exclude hidden files
    • Follow symbolic links: Choose whether to follow symlinks

2. Selection Strategy

Choose which files to keep when duplicates are found:

  • Keep newest file: Preserves the most recently modified file in each group
  • Keep oldest file: Preserves the earliest created file in each group
  • Keep shortest path: Keeps the file with the shortest directory path
  • Keep longest path: Keeps the file with the longest directory path

The strategy can be changed at any time, even after scanning, to automatically update selections.

3. Scanning Process

  1. Click "Select folder" to choose the directory to scan
  2. Review the selected directory path
  3. Click "Start Scan" to begin the duplicate search
  4. Wait for the scan to complete - progress will be shown

4. Managing Results

  • View Results: Switch to the "RESULTS" tab after scanning
  • Group Information: Each group shows:
    • Number of duplicate files
    • File size
    • Modification dates
    • Full file paths
  • Selection Controls:
    • "Select All": Choose all duplicate files
    • "Select None": Clear all selections
    • Individual checkboxes: Manually select specific files
  • Deletion:
    • Review selected files carefully
    • Click "Delete Selected" to remove chosen duplicates
    • Confirm deletion when prompted

Tips for Effective Use

  • Start with smaller directories to familiarize yourself with the interface
  • Use appropriate file type filters to focus on specific files
  • Review the space savings shown before deletion
  • Change the selection strategy to match your organization preferences

Important Notes about Cloud Storage

  • Files must be downloaded locally before scanning
  • Deleting files from cloud storage folders affects both local and cloud copies
  • Changes will sync across all connected devices
  • Consider moving files to a local folder before deletion to prevent unintended synchronization

Troubleshooting

  • No duplicates found: Check file type filters and minimum size settings
  • Scan takes too long: Try scanning smaller directories or using more specific file types
  • Permission errors: Ensure you have access to all directories being scanned
