Duplicate Files Cleanup Utility - Instructions

Getting Started

This utility helps you find and manage duplicate files in your system. It's powered by fclones, a high-performance duplicate file finder.

Installation

Windows

  1. Install Rust (if not already installed): curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  2. Install fclones: cargo install fclones

Linux

cargo install fclones

macOS

brew install fclones

Using the Interface

1. Select Directory

Choose the directory you want to scan for duplicates. You can enter the path manually or use the file picker.

2. Configure Scan

  • Hashing Algorithm: Choose the algorithm for file comparison
    • md5: Fastest, good for general use
    • sha1: More secure, slightly slower
    • sha256: Most secure, slower
    • xxhash: Extremely fast, less secure
    • blake3: Fast and secure
  • Size Limits: Set minimum and maximum file sizes to scan
    • Use suffixes: K (kilobytes), M (megabytes), G (gigabytes)
    • Example: 1M, 100K, 1G
  • Threads: Number of parallel processing threads

3. Advanced Options

  • Exclude Pattern: Skip files matching the pattern
    • Example: *.git to exclude Git directories
    • Example: *.tmp to exclude temporary files
  • Output Format: Choose how to view results
    • text: Simple text output
    • json: Structured JSON data
    • csv: Tabular format for spreadsheet import

4. Running the Scan

  1. Click "Start Scan" to begin
  2. Wait for the scan to complete
  3. Review the results in the "Results" tab

Tips for Best Results

  • Start with smaller directories to test the configuration
  • Use faster algorithms (md5 or xxhash) for initial scans
  • Adjust thread count based on your system's capabilities
  • Use size limits to focus on important files

Important Notes

  • Always review results before taking any action
  • System directories are protected by default
  • Large scans may take significant time and resources
  • Keep backups of important files

Troubleshooting

  • Scan fails: Check if fclones is properly installed
  • Permission errors: Run with appropriate permissions
  • Memory issues: Reduce thread count or scan size
